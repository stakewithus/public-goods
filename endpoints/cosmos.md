### RPC / GRPC / LCD ###

StakeWithUs is currently providing public rpc/grpc/lcd endpoints for the following projects:

1) Cosmos (ATOM)
2) Osmosis (OSMOS)
3) Injective (INJ)
4) Comdex (CMDX)
5) Persistence (XPRT)
6) Bandchain (BAND)
7) Juno (JUNO)
8) Evmos (EVMOS)
9) Gravity (GRAVITON)
10) Kava (KAVA)
11) Akash (AKASH)
12) Terra (LUNA)
13) Agoric (BLD)
14) QuickSilver (QCK)
15) Celer (CELER)
16) Tgrade (TGRADE)
17) Starname (IOV)
18) Passage (PASG)
19) Kujira (KUJI)

### Rate Limits ###

Rate limit are set to 200 requests per minute for all endpoints
