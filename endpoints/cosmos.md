### RPC / GRPC / LCD

StakeWithUs is currently providing public rpc/grpc/lcd endpoints for the following projects:

1. Agoric (BLD)
2. Akash (AKASH)
3. Archway (ARCH)
4. Bandchain (BAND)
5. Canto (CANTO)
6. Celer (CELER)
7. Celestia (TIA)
8. Cosmos (ATOM)
9. dYdX (DYDX)
10. Injective (INJ)
11. Kava (KAVA)
12. Kujira (KUJI)
13. Osmosis (OSMOS)
14. Passage (PASG)
15. Persistence (XPRT)
16. QuickSilver (QCK)
17. Terra (LUNA)

### Rate Limits

Rate limit are set to 200 requests per minute for all endpoints
