### RPC / GRPC / LCD ###

StakeWithUs is currently providing public rpc/grpc/lcd endpoints for the following projects:

1) Cosmos (ATOM)
2) Osmosis (OSMOS)
3) Injective (INJ)
4) Persistence (XPRT)
5) Bandchain (BAND)
6) Kava (KAVA)
7) Akash (AKASH)
8) Terra (LUNA)
9) Agoric (BLD)
10) QuickSilver (QCK)
11) Celer (CELER)
12) Passage (PASG)
13) Kujira (KUJI)
14) Celestia (TIA)

### Rate Limits ###

Rate limit are set to 200 requests per minute for all endpoints
