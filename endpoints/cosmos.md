### RPC / GRPC / LCD

StakeWithUs is currently providing public rpc/grpc/lcd endpoints for the following projects:

1. Agoric (BLD)
2. Akash (AKASH)
3. Archway (ARCH)
4. Bandchain (BAND)
5. Celer (CELER)
6. Celestia (TIA)
7. Certik Shentu (CTK)
8. Cosmos (ATOM)
9. dYdX (DYDX)
10. Kava (KAVA)
11. Osmosis (OSMOS)
12. Passage (PASG)
13. Persistence (XPRT)
14. Selfchain (SLF)
15. Terra (LUNA)

### Rate Limits

Rate limit are set to 200 requests per minute for all endpoints
